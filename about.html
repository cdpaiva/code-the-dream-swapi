About the characters images

I also wanted to display images of the characters, although the API didn't provide that. To do so, I found a page that had images of several star wars characters and used ChapGPT to get a script that scraped the page and saved all images. I knew that I could do that with a python library called beautifulsoup, but I didn't quite know how, so I had ChatGPT to generate the code for me. The code is in the file scrape.py.

Then, I used the API to count in how many films each character appeared. If the characters appeared in 3 movies or more, I gave them their images. The other ones don't have the images, since my goal was more to figure out how to do it than to add for all the 82 characters provided by the API.

Posters for each movie were downloaded from https://www.allposters.com
